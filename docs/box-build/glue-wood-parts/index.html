Box assembly Step 2: Glue

Step 2: Glue
First set out the box pieces like this.

Donâ€™t include the top piece with the hole in it. The kit is easiest to assemble and maintain if this top piece is never glued in.
Make sure the sides you want on the inside are facing up.

Now put glue on the cracks.
Put glue on all the joint fingers except the six for the box top.
Now fold up the sides.
Squeeze the parts together tightly!
Set the box aside to dry.
Do the same thing with the lid.
Now glue the motor mounts together.
Again, make sure the parts are squeezed together tightly.
When the box is dry, glue the first piece of the battery holder holder.
Make sure to put the battery holder holder in the shorter lid-end of the box.
Next, glue the thin hinge pieces in.
They should fit into the corners with the battery holder holder.
Now put the other part of the battery holder holder in.
Finally, glue the top hinge piece in the lid.

The gluing is done! Let everything dry and start assembling the electrical parts.
